CYCLOID (EPG Soluction)
==================================================================

- Clone branch in your local. https://github.com/thiagoscamilo/cycloid
- Run following mvn install 
- Run application
- Open Swagger - http://localhost:8080/swagger-ui/index.html

Frameworks

- Java 8
- Spring boot 2 with H2 database
- Maven
- Spring Validation
- Swagger
- Junit 5
